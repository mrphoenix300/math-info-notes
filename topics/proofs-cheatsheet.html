Cheatsheet Τεχνικών Απόδειξης

Quick reference για όλες τις τεχνικές που χρησιμοποιούνται στο μάθημα. Πότε, πώς, παράδειγμα, παγίδες.

1. Άμεση Απόδειξη

➡️

Άμεση Απόδειξη

Πότε

Όταν οι υποθέσεις μεταφράζονται απευθείας στο συμπέρασμα μέσω γνωστών θεωρημάτων, ορισμών ή αλγεβρικών μετασχηματισμών.

Φόρμα
Υποθέτω \(P\). Με βάση [ορισμό/θεώρημα]: ... Άρα \(Q\). \(\square\)
Παράδειγμα
«Αν \(n\) ζυγός, τότε \(n^2\) ζυγός.» Βάζω \(n = 2k\), τότε \(n^2 = 4k^2 = 2(2k^2)\).
  • Μην παρουσιάζετε ως «άμεση» μια απόδειξη που κρύβει κυκλικό συλλογισμό.
  • Φροντίστε να χρησιμοποιείτε ρητά τις υποθέσεις — μην τις αγνοείτε.

2. Αντιθετοαντίστροφη (Contrapositive)

↩️

Αντιθετοαντίστροφη

Πότε

Όταν το «αν \(\neg Q\) τότε \(\neg P\)» είναι φυσικότερο να αποδείξετε. Ισοδύναμο λογικά με \(P \Rightarrow Q\).

Φόρμα
Αποδεικνύω \(\neg Q \Rightarrow \neg P\):
Υποθέτω \(\neg Q\). ... Άρα \(\neg P\). \(\square\)
Παράδειγμα
«Αν \(n^2\) ζυγός τότε \(n\) ζυγός.» Αντιστρόφως: αν \(n\) μονός (\(n=2k+1\)) τότε \(n^2=4k^2+4k+1\) μονός.
  • Μη μπερδεύετε με «αντίστροφο» (\(Q \Rightarrow P\)) — είναι διαφορετική πρόταση.
  • Πρέπει να αποδείξετε \(\neg Q \Rightarrow \neg P\), όχι \(\neg P \Rightarrow \neg Q\).

3. Απαγωγή σε Άτοπο (Contradiction)

💥

Απαγωγή σε Άτοπο

Πότε

Όταν δεν μπορείτε να κατασκευάσετε αντικείμενο με τις ιδιότητες \(\neg P\), ή όταν \(\neg P\) οδηγεί αναπόφευκτα σε αντίφαση.

Φόρμα
Υποθέτω \(\neg P\). Τότε ... [αλυσίδα συλλογισμών] ... Άρα \(A \land \neg A\). Άτοπο. Άρα \(P\). \(\square\)
Παράδειγμα
«\(\sqrt{2}\) άρρητος.» Υποθέτω \(\sqrt{2} = p/q\) πλήρως απλοποιημένο. Τότε \(2q^2 = p^2\) → \(p\) ζυγός → \(p=2k\) → \(q^2=2k^2\) → \(q\) ζυγός. Αντίφαση.
  • Η αντίφαση πρέπει να πηγάζει από \(\neg P\) — μην χρησιμοποιείτε ήδη αποδεδειγμένα γεγονότα ως «άτοπο».
  • Προτιμήστε αντιθετοαντίστροφη όταν είναι δυνατό (πιο διαφανής).

4. Επαγωγή (Μαθηματική / Ισχυρή / Δομική)

🔁

Επαγωγή

Πότε

Ιδιότητες για φυσικούς αριθμούς ή επαγωγικά ορισμένες δομές. Βλ. αναλυτικά Οδηγό Επαγωγής.

Απλή
Βήμα \(n \to n+1\), Ε.Υ. για \(n\)
Ισχυρή
Ε.Υ. για κάθε \(m < n\)
Δομική
Ε.Υ. για «μικρότερα» υπο-αντικείμενα
Παράδειγμα
Φόρμουλες αθροισμάτων (απλή), Fibonacci (ισχυρή), \(\ell(x\circ y)\) (δομική)
  • Λάθος βάση → όλη η απόδειξη καταρρέει.
  • Κυκλική Ε.Υ. (χρήση \(n+1\) για \(n+1\)) — μη-αποδεκτό.

5. Κατασκευαστική Απόδειξη Ύπαρξης

🔨

Κατασκευαστική Ύπαρξη

Πότε

Όταν η εκφώνηση λέει «υπάρχει» και μπορείτε να κατασκευάσετε ρητά αυτό που ζητείται.

Φόρμα
«Ορίζουμε \(x = \ldots\). Αποδεικνύουμε ότι \(x\) έχει τις ζητούμενες ιδιότητες.»
Παράδειγμα
«Υπάρχουν \(x, y \in \Z\) με \(ax + by = \gcd(a,b)\).» Κατασκευή: ExtGCD αλγόριθμος (Δ4).
  • Δεν φτάνει να «πείτε» ότι υπάρχει — πρέπει να κατασκευάσετε ρητά ή να αποδείξετε ύπαρξη με άλλη μέθοδο.
  • Βεβαιωθείτε ότι η κατασκευή σας ικανοποιεί όλους τους απαιτούμενους περιορισμούς.
Μη-Κατασκευαστική Ύπαρξη — \(\sqrt{2}^{\sqrt{2}}\)

Ορίζουμε \(a = \sqrt{2}^{\sqrt{2}}\). Αν \(a\) άρρητος: τότε \(a^{\sqrt{2}} = \sqrt{2}^2 = 2\) ρητός — άρα \(x=a, y=\sqrt{2}\) δίνει \(x^y\) ρητό. Αν \(a\) ρητός: τότε \(x = y = \sqrt{2}\) δίνει \(x^y = a\) ρητό. Σε κάθε περίπτωση ο ισχυρισμός ισχύει, χωρίς να γνωρίζουμε ποια περίπτωση ισχύει! (Πηγή: Δ4 §1.)

6. Αντιπαράδειγμα

❌

Αντιπαράδειγμα

Πότε

Για να αποδείξετε ότι ένας ∀-ισχυρισμός είναι ψευδής: αρκεί ένα μόνο αντιπαράδειγμα.

Φόρμα
«Ο ισχυρισμός είναι ψευδής. Αντιπαράδειγμα: \(x = \ldots\). Τότε [επαληθεύστε ότι \(\neg P(x)\)].»
Παράδειγμα
«Κάθε πρώτος είναι μονός.» Αντιπαράδειγμα: \(2\) είναι πρώτος και ζυγός.
  • Ένα αντιπαράδειγμα αρκεί — δεν χρειάζεστε απαρίθμηση.
  • Μην δίνετε «ψευδο-αντιπαράδειγμα» — ελέγξτε ότι πράγματι ικανοποιεί τις υποθέσεις αλλά όχι το συμπέρασμα.

7. Αρχή Περιστερεώνα (Pigeonhole)

🐦

Αρχή Περιστερεώνα

Πότε

Όταν έχετε \(n+1\) αντικείμενα και \(n\) κατηγορίες — κάποια κατηγορία έχει \(\ge 2\). Γενίκευση: αν \(m\) αντικείμενα σε \(n\) κλάσεις, τότε κάποια έχει \(\ge \lceil m/n \rceil\).

Φόρμα
«Ορίζουμε \(n\) "συρτάρια" ως [...].
Αφού \(m > n\), κάποιο συρτάρι περιέχει \(\ge 2\) αντικείμενα.»
Παράδειγμα
Σε 13 ανθρώπους υπάρχουν δύο με ίδιο μήνα γέννησης. (13 άνθρωποι, 12 μήνες.)
  • Οι κλάσεις πρέπει να καλύπτουν ΟΛα τα αντικείμενα.
  • Μην κάνετε την ανάθεση κυκλική — πρέπει να είναι σαφής.

Εφαρμογές στο Μάθημα

ΕφαρμογήΑντικείμεναΣυρτάριαΠηγή
Dirichlet Approximation \(m+1\) κλασματικά μέρη \(\{k\theta\}\) \(m\) διαστήματα \([i/m,(i+1)/m)\) Δ4 §6
Erdős–Szekeres \(n^2+1\) αριθμοί, \(d_i\) αύξουσες υπακολουθίες \(n\) τιμές \(1,\ldots,n\) Δ4 §5

8. Πιθανοτική Μέθοδος

🎲

Πιθανοτική Μέθοδος

Πότε

Για να αποδείξετε ύπαρξη αντικειμένου με κάποια ιδιότητα, χωρίς να το κατασκευάσετε ρητά. Αν σε τυχαία επιλογή \(\Pr[\text{ιδιότητα}] > 0\), τότε το αντικείμενο υπάρχει.

Φόρμα (Union Bound)
Αν \(\Pr[\exists \text{ "κακό" } A] \le \sum \Pr[A_i] < 1\), τότε με θετική πιθανότητα κανένα κακό δεν συμβαίνει.
Παράδειγμα
Erdős (1947): \(R(k,k) > 2^{k/2}\). Τυχαίος 2-χρωματισμός \(K_N\): \(\Pr[\text{υπάρχει mono } K_k] < 1\) για \(N = 2^{k/2}\).
  • Μη-κατασκευαστική: δεν λέμε ποιο αντικείμενο — μόνο ότι υπάρχει.
  • Προσέξτε ότι το σύμπαν πιθανότητας είναι καλά ορισμένο.
  • Union bound: \(\Pr[\bigcup A_i] \le \sum \Pr[A_i]\) — ισχύει αλλά μπορεί να είναι χαλαρό.

Εφαρμογές στο Μάθημα

ΕφαρμογήΤυχαία ΕπιλογήΑποτέλεσμαΠηγή
\(m(n) \ge 2^{n-1}\) Τυχαία 2-χρώμ. \(K_n\) Λίγες μονόχρωμες \(K_3\) Δ4 §3
\(R(k,k) > 2^{k/2}\) Τυχαίος 2-χρωμ. \(K_N\) Δεν υπάρχει mono \(K_k\) με πιθανότητα \(> 0\) Δ4 §4

9. Διαγωνοποίηση

📐

Διαγωνοποίηση (Cantor)

Πότε

Για να αποδείξετε μη-απαριθμησιμότητα ή μη-αποφασιστότητα. Κατασκευάστε αντικείμενο που διαφέρει από το \(i\)-οστό στην \(i\)-οστή συντεταγμένη.

Φόρμα
Υποθέτω απαρίθμηση \(x_1, x_2, \ldots\).
Κατασκευώ \(y\) με \(y_i \ne (x_i)_i\).
Τότε \(y \ne x_i\) για κάθε \(i\) — άτοπο.
Παράδειγμα 1
\(\R\) μη-απαριθμήσιμο (Cantor 1874). Δεκαδικό \(y_i = \begin{cases}5 & \text{αν } (x_i)_i \ne 5 \\ 6 & \text{αν } (x_i)_i = 5\end{cases}\).
Παράδειγμα 2
Halting Problem: Πρόγραμμα \(D\) που «εκτελείται σε εαυτό και αντιστρέφει».
  • Η επιλογή \(y_i\) πρέπει να είναι ρητά ορισμένη και να αποφεύγει συγκεκριμένες τιμές (π.χ. αποφύγετε 0 και 9 για να μην υπάρξει 0.999... = 1.000... πρόβλημα).
  • Για Halting: ο \(D\) πρέπει να είναι έγκυρο πρόγραμμα — επαληθεύστε ότι η κατασκευή είναι συνεπής.
Διαγράμμα Διαγωνοποίησης (Halting Problem)

Πίνακας \(G[i][j] = 1\) αν πρόγραμμα \(P_i\) τερματίζει με είσοδο \(\langle P_j \rangle\):

      P₁  P₂  P₃  P₄  ···
P₁ [  1    0    1    1  ···]
P₂ [  0    1    0    0  ···]
P₃ [  1    1    0    1  ···]  ← διαγώνιος: G[1,1],G[2,2],G[3,3],...
P₄ [  0    0    1    1  ···]
 D [  0    1    1    0  ···]  ← D: αντίθετο της διαγωνίου

Πρόγραμμα \(D\): «αν \(G[i,i]=1\) τότε μη-τερμάτισε, αλλιώς τερμάτισε». Για \(i = D\): \(G[D,D] = 1 \iff D\) τερματίζει \(\iff G[D,D]=0\). Αντίφαση. \(\square\)

Πηγή: Δ5 §5
