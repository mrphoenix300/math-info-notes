2. Κεντρικά Θεωρήματα

Λήμμα Χειραψίας

\[\sum_{v \in V} \deg(v) = 2|E|\]

Ο αριθμός κόμβων περιττού βαθμού είναι ζυγός.

🔗 Δ6 §Α

Αριθμοί Ramsey

\[R(k,m) \le R(k-1,m) + R(k,m-1), \quad R(k,k) < 4^k\]

Σε οποιοδήποτε \(K_{R(k,m)}\), υπάρχει \(K_k\) μονόχρωμο κόκκινο ή \(K_m\) μπλε.

🔗 Δ6 §Α · Δ2 §3

Θεώρημα Δέντρων (Ισοδύναμες Συνθήκες)

Για συνεκτικό \(G\) με \(n\) κόμβους, ισοδυναμούν:

  1. \(G\) είναι δέντρο (συνεκτικός ακυκλικός)
  2. \(G\) έχει ακριβώς \(n-1\) ακμές
  3. Κάθε ζεύγος κόμβων συνδέεται με μοναδικό μονοπάτι
  4. Αφαίρεση οποιασδήποτε ακμής αποσυνδέει
  5. Προσθήκη οποιασδήποτε ακμής δημιουργεί κύκλο
🔗 Δ6 §Β

Θεώρημα Euler (Euler Κύκλος)

Συνεκτικός \(G\) έχει Euler κύκλο \(\iff\) κάθε κόμβος έχει ζυγό βαθμό.

Euler μονοπάτι: ακριβώς δύο κόμβοι περιττού βαθμού.

🔗 Δ6 §Γ

Τύπος Euler (Επίπεδα Γραφήματα)

\[n - m + f = 2\]

Για συνεκτικό επίπεδο γράφο: \(n\) κόμβοι, \(m\) ακμές, \(f\) πρόσωπα.

🔗 Δ6 §Δ

Φράγμα Αραιότητας

Επίπεδος γράφος με \(n \ge 3\): \(m \le 3n-6\).
Διμερής επίπεδος: \(m \le 2n-4\).
🔗 Δ6 §Δ

Θεώρημα Kuratowski / Wagner

\(G\) επίπεδος \(\iff\) δεν περιέχει subdivision του \(K_5\) ή \(K_{3,3}\).
🔗 Δ6 §Δ

Θεώρημα Hall

Διμερής \(G=(L\cup R, E)\) έχει τέλειο ταίριασμα του \(L\) \(\iff\) \(\forall S \subseteq L: |N(S)| \ge |S|\).
🔗 Δ7 §4

Θεώρημα Dilworth

\[c(P) = \alpha(P)\]

Ελάχιστη κάλυψη αλυσίδων = μέγιστη αντιαλυσίδα σε κάθε poset.

🔗 Δ8 §7

3. Decision Tree: Ποιο Θεώρημα Χρησιμοποιώ;

«Δείξτε ότι σε κάθε γράφο υπάρχουν δύο κόμβοι ίδιου βαθμού» Αρχή Περιστερεώνα

Οι βαθμοί είναι στο \(\{0, 1, \ldots, n-1\}\) — \(n\) τιμές για \(n\) κόμβους, αλλά δεν μπορεί ταυτόχρονα να υπάρχουν βαθμός 0 και βαθμός \(n-1\). Άρα το πολύ \(n-1\) διακριτές τιμές για \(n\) κόμβους.

«Αποδείξτε ότι \(\sum_{v}\deg(v)\) είναι ζυγός / μετρήστε ακμές» Λήμμα Χειραψίας

\(\sum \deg(v) = 2|E|\).

«Υπάρχει ή δεν υπάρχει Euler κύκλος/μονοπάτι;» Θεώρημα Euler

Μετρήστε κόμβους περιττού βαθμού: 0 → Euler κύκλος, 2 → Euler μονοπάτι, άλλο → κανένα.

«Αποδείξτε/ελέγξτε ότι γράφος είναι δέντρο» Θεώρημα Δέντρων

Αρκεί: συνεκτικός + \(n-1\) ακμές. Ή: συνεκτικός + ακυκλικός. Ή: μοναδικά μονοπάτια.

«Αποδείξτε/ελέγξτε επιπεδότητα γράφου» Τύπος Euler + Φράγμα + Kuratowski

Για απόδειξη μη-επιπεδότητας: αν \(m > 3n-6\) ή βρείτε \(K_5\)/\(K_{3,3}\) ως minor/subdivision. Για απόδειξη επιπεδότητας: σχεδιάστε χωρίς τομές.

«Υπάρχει τέλειο ταίριασμα σε διμερή γράφο;» Θεώρημα Hall

Ελέγξτε \(|N(S)| \ge |S|\) για κάθε \(S \subseteq L\). Αν βρείτε \(S\) με \(|N(S)| < |S|\): Hall violation, δεν υπάρχει.

«Ελάχιστη κάλυψη αλυσίδων / μέγιστη αντιαλυσίδα» Θεώρημα Dilworth

Κατασκεύασε διμερή γράφο \(H\), βρες μέγιστο ταίριασμα, κάλυψη = \(n - |M_\max|\).

«Σε κάθε ακολουθία \(n^2+1\) αριθμών, υπάρχει μονότονη υπακολουθία μήκους \(n+1\)» Erdős–Szekeres (Dilworth)

Ορίζουμε poset, εφαρμόζουμε Dilworth ή Mirsky.

«Υπάρχει μονόχρωμο \(K_k\) σε 2-χρωματισμό \(K_N\)» Αριθμοί Ramsey / Πιθανοτική Μέθοδος

Για \(N = R(k,k)\): σίγουρα. Για \(N < 2^{k/2}\): πιθανοτική μέθοδος δείχνει ότι ενδέχεται να μην υπάρχει.

4. Συνηθισμένοι Τύποι Ασκήσεων

Τύπος Α — Αποδείξτε / Διαψεύστε ιδιότητα γράφου

Στρατηγική:

  1. Αν χρειάζεται ύπαρξη: Ramsey, Hall, περιστερεώνας, πιθανοτική.
  2. Αν χρειάζεται αναγκαία συνθήκη: χειραψία, Euler, φράγμα.
  3. Αν ζητείτε παράδειγμα/αντιπαράδειγμα: δοκιμάστε \(K_n\), \(C_n\), \(K_{m,n}\), \(P_n\).
Τύπος Β — Υπολογίστε \(\alpha(G)\), \(\omega(G)\), \(\chi(G)\)

Τρικ:
— \(\chi(G) \ge \omega(G)\) (κλίκες χρειάζονται διαφορετικά χρώματα).
— Για διμερείς: \(\chi(G) \le 2\).
— Για επίπεδους: \(\chi(G) \le 4\) (Θεώρημα 4 Χρωμάτων), \(\le 6\) εύκολα.
— \(\alpha(G) \cdot \chi(G) \ge n\) (κάθε χρώμα = ανεξάρτητο σύνολο).

Τύπος Γ — Δείξτε ότι γράφος δεν είναι επίπεδος

Βήματα:
1. Ελέγξτε \(m \le 3n-6\) (αν παραβιάζεται: μη-επίπεδος).
2. Αν διμερής: \(m \le 2n-4\).
3. Βρείτε subdivision \(K_5\) ή \(K_{3,3}\) (Kuratowski).
Παράδειγμα: \(K_5\): \(n=5, m=10 > 3\cdot5-6=9\). Άρα μη-επίπεδος.

Τύπος Δ — Hall / Ταίριασμα σε Εφαρμογή

Βήματα:
1. Μοντελοποιήστε ως διμερή γράφο (L = «αντικείμενα», R = «θέσεις»/«ζεύγη»).
2. Ελέγξτε Hall condition.
3. Αν Hall παραβιάζεται, βρείτε \(S \subseteq L\) με \(|N(S)| < |S|\).

Τύπος Ε — Ramsey Lower/Upper Bound

Upper: \(R(k,m) \le R(k-1,m)+R(k,m-1)\) — επαγωγή στο \(k+m\).
Lower: Πιθανοτική μέθοδος (Erdős 1947) — τυχαίος 2-χρωματισμός \(K_N\) για \(N = 2^{k/2}\) δεν έχει μονόχρωμο \(K_k\) με θετική πιθανότητα.
