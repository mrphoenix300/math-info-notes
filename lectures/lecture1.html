Διάλεξη 1

Διάλεξη 1 — Υποθέσεις, Εικασίες, Θεωρήματα

TL;DR
  • Η μαθηματική γνώση χτίζεται ιεραρχικά: αξιώματα → ορισμοί → λήμματα → θεωρήματα.
  • Μια υπόθεση γίνεται θεώρημα μόνο με απόδειξη· χωρίς αυτήν παραμένει εικασία — ακόμα κι αν επαληθεύεται σε εκατομμύρια περιπτώσεις.
  • Αρκεί ένα αντιπαράδειγμα για να καταρρίψει μια υπόθεση.
  • Το Θεώρημα Beatty: οι ακολουθίες \(\lfloor k\varphi \rfloor\) και \(\lfloor k\varphi^2 \rfloor\) διαμερίζουν τους θετικούς ακεραίους.
  • Το P vs NP παραμένει το κορυφαίο ανοιχτό πρόβλημα Πληροφορικής· \(\text{SAT}\) είναι NP-πλήρες (Cook-Levin).

1. Υπόθεση, Εικασία, Θεώρημα

Πριν μιλήσουμε για αποδείξεις, χρειαζόμαστε ένα κοινό λεξιλόγιο. Τα μαθηματικά λειτουργούν σαν ένα οικοδόμημα: χτίζουμε νέες αλήθειες πάνω σε παλιές, ακολουθώντας αυστηρή ιεραρχία.

Ορισμός — Αξίωμα

Ένα αξίωμα (axiom) είναι μια πρόταση που δεχόμαστε ως αληθής χωρίς απόδειξη. Αποτελεί τη θεμέλιο λίθο κάθε μαθηματικής θεωρίας.

Παράδειγμα (Ευκλείδης): «Από δύο σημεία διέρχεται ακριβώς μία ευθεία.»

Ορισμός — Ορισμός, Λήμμα, Θεώρημα, Πόρισμα
  • Ορισμός (definition): Δίνει ακριβές νόημα σε μια μαθηματική έννοια.
  • Λήμμα (lemma): Βοηθητικό αποτέλεσμα που χρησιμοποιείται για να αποδείξουμε κάτι μεγαλύτερο.
  • Θεώρημα (theorem): Σημαντική πρόταση που έχει αποδειχθεί.
  • Πόρισμα (corollary): Αποτέλεσμα που προκύπτει άμεσα από θεώρημα.

Η ροή της μαθηματικής έρευνας είναι συνήθως:

Ορισμός — Εικασία

Μια εικασία (conjecture) είναι μια πρόταση για την οποία:

  • υπάρχει ισχυρή αριθμητική ή άλλη ένδειξη ότι ισχύει,
  • αλλά δεν έχει αποδειχθεί ούτε έχει βρεθεί αντιπαράδειγμα.

Αρκεί ένα μόνο αντιπαράδειγμα για να τη διαψεύσουμε. Αντίθετα, δεν αρκεί πεπερασμένο πλήθος επαληθεύσεων για να τη «αποδείξουμε».

Προσοχή

Το γεγονός ότι μια πρόταση ισχύει για τις πρώτες 10, 1000 ή \(10^{100}\) περιπτώσεις δεν αποτελεί απόδειξη. Θα δούμε παρακάτω ότι το \(n^2 + n + 41\) είναι πρώτος για τα πρώτα 40 θετικά ακέραια — αλλά όχι για το 41ο.

2. Η Χρυσή Τομή \(\varphi\)

Η χρυσή τομή εμφανίζεται σε γεωμετρία, τέχνη, φυσική και — ενδιαφέρον για εμάς — σε αριθμοθεωρητικές κατανομές των φυσικών αριθμών.

Ορισμός — Χρυσή Τομή

Η χρυσή τομή ορίζεται ως η θετική λύση της εξίσωσης \(x^2 = x + 1\):

\[ \varphi = \frac{1 + \sqrt{5}}{2} \approx 1{,}6180339\ldots \]

2.1 Βασικές Ιδιότητες του \(\varphi\)

Θεώρημα — Ιδιότητες Χρυσής Τομής
  1. \(\varphi^2 = \varphi + 1\)
  2. \(\dfrac{1}{\varphi} = \varphi - 1 = \dfrac{\sqrt{5}-1}{2} \approx 0{,}618\)
  3. \(\dfrac{1}{\varphi} + \dfrac{1}{\varphi^2} = 1\)
Απόδειξη ιδιοτήτων

(1) Άμεση επαλήθευση:

\[ \varphi^2 = \left(\frac{1+\sqrt{5}}{2}\right)^2 = \frac{1 + 2\sqrt{5} + 5}{4} = \frac{6 + 2\sqrt{5}}{4} = \frac{3 + \sqrt{5}}{2} = \frac{1 + \sqrt{5}}{2} + 1 = \varphi + 1. \]

(2) Από \(\varphi^2 = \varphi + 1\) διαιρούμε δι' \(\varphi\): \(\varphi = 1 + 1/\varphi\), άρα \(1/\varphi = \varphi - 1 = (\sqrt{5}-1)/2\).

(3) \[ \frac{1}{\varphi} + \frac{1}{\varphi^2} = \frac{\varphi + 1}{\varphi^2} = \frac{\varphi^2}{\varphi^2} = 1, \] χρησιμοποιώντας \(\varphi + 1 = \varphi^2\) από (1). \(\qquad\square\)

Παράδειγμα — Πρώτες τιμές

Υπολογίζουμε τα \(\lfloor k\varphi \rfloor\) και \(\lfloor k\varphi^2 \rfloor\) για \(k = 1, 2, \ldots, 8\):

Οι τιμές \(\{1,3,4,6,8,9,11,12,\ldots\}\) και \(\{2,5,7,10,13,15,18,20,\ldots\}\) είναι ξένες μεταξύ τους και καλύπτουν μαζί τους \(\mathbb{N}^+\). Αυτό είναι το Θεώρημα Beatty.

2.2 Θεώρημα Beatty

Το θεώρημα Beatty γενικεύει την παρατήρηση για \(\varphi\) σε οποιαδήποτε ζεύγη αρρήτων αριθμών με άθροισμα αντίστροφων = 1.

Θεώρημα Beatty (1926)

Έστω \(r, s > 1\) θετικοί άρρητοι αριθμοί με \[\frac{1}{r} + \frac{1}{s} = 1.\] Τότε οι ακολουθίες \[A = \{\lfloor r \rfloor, \lfloor 2r \rfloor, \lfloor 3r \rfloor, \ldots\} \quad \text{και} \quad B = \{\lfloor s \rfloor, \lfloor 2s \rfloor, \lfloor 3s \rfloor, \ldots\}\] αποτελούν διαμέριση του \(\mathbb{N}^+ = \{1, 2, 3, \ldots\}\), δηλαδή \(A \cap B = \emptyset\) και \(A \cup B = \mathbb{N}^+\).

Εφαρμογή για \(\varphi\): Παίρνουμε \(r = \varphi\), \(s = \varphi^2 = \varphi + 1\), και επαληθεύουμε \(1/\varphi + 1/\varphi^2 = 1\) (ιδιότητα 3 παραπάνω).

Απόδειξη Θεωρήματος Beatty

Αποδεικνύουμε τα δύο μέρη ξεχωριστά.

Βήμα 1: \(A \cap B = \emptyset\).

Έστω, για αντίφαση, \(\lfloor pr \rfloor = \lfloor qs \rfloor = m\) για κάποιους \(p, q \in \mathbb{N}^+\). Επειδή \(r\) και \(s\) είναι άρρητοι, τα \(pr\) και \(qs\) δεν είναι ακέραιοι, οπότε:

\[ m < pr < m+1 \quad \text{και} \quad m < qs < m+1. \]

Διαιρώντας:

\[ \frac{m}{r} < p < \frac{m+1}{r} \quad \text{και} \quad \frac{m}{s} < q < \frac{m+1}{s}. \]

Προσθέτουμε τις ανισότητες:

\[ m\!\left(\frac{1}{r}+\frac{1}{s}\right) < p+q < (m+1)\!\left(\frac{1}{r}+\frac{1}{s}\right) \;\Longrightarrow\; m < p + q < m + 1. \]

Άτοπο, αφού \(p+q\) είναι ακέραιος.

Βήμα 2: \(A \cup B = \mathbb{N}^+\).

Ορίζουμε \(A_n = \{a \in A : a \le n\}\) και \(B_n = \{b \in B : b \le n\}\). Θα δείξουμε \(|A_n| + |B_n| = n\).

Παρατηρούμε:

\[ |A_n| = \#\{k \ge 1 : \lfloor kr \rfloor \le n\} = \#\{k \ge 1 : kr < n+1\} = \#\{k \ge 1 : k < \tfrac{n+1}{r}\} = \left\lfloor \frac{n+1}{r} \right\rfloor, \]

όπου χρησιμοποιήσαμε ότι \((n+1)/r\) είναι άρρητος (άρα ουδέποτε ακέραιος), και ομοίως \(|B_n| = \lfloor (n+1)/s \rfloor\).

Θέτουμε \(M = n+1\). Αρκεί να δείξουμε \(\lfloor M/r \rfloor + \lfloor M/s \rfloor = M-1\). Έχουμε:

\[ \frac{M}{r} + \frac{M}{s} = M\!\left(\frac{1}{r}+\frac{1}{s}\right) = M. \]

Άρα \(\lfloor M/r\rfloor + \lfloor M/s\rfloor = M - \bigl(\{M/r\} + \{M/s\}\bigr)\). Το άθροισμα κλασματικών μερών πρέπει να είναι ακέραιος (η αριστερά πλευρά είναι ακέραιος). Επειδή \(r,s\) είναι άρρητοι: \(0 < \{M/r\} < 1\) και \(0 < \{M/s\} < 1\), άρα \(\{M/r\} + \{M/s\} = 1\), και:

\[ |A_n| + |B_n| = \left\lfloor\frac{n+1}{r}\right\rfloor + \left\lfloor\frac{n+1}{s}\right\rfloor = (n+1) - 1 = n. \]

Εφόσον \(A_n, B_n\) ξένα, περιέχουν αριθμούς \(\le n\), και \(|A_n|+|B_n|=n\), συνάγεται \(A_n \cup B_n = \{1,\ldots,n\}\) για κάθε \(n\).

Συχνή ερώτηση εξετάσεων

Το Θεώρημα Beatty χρησιμοποιεί μόνο τις ιδιότητες:

  1. \(1/r + 1/s = 1\) (αρκεί ακριβώς αυτό)
  2. \(r\) και \(s\) είναι άρρητοι (ώστε \(\lfloor kr \rfloor\) και \(kr\) να μη συμπίπτουν)

Το ότι \(r = \varphi\) δεν παίζει ειδικό ρόλο στην απόδειξη. Μπορείτε να δώσετε άλλο ζεύγος; Π.χ. \(r = \sqrt{2}\), \(s = 2 + \sqrt{2}\).

3. Κατάρριψη Υπόθεσης — Αντιπαράδειγμα

Ένα αντιπαράδειγμα είναι μια συγκεκριμένη περίπτωση στην οποία η υπόθεση αποτυγχάνει. Ένα μόνο αρκεί για να την ανατρέψει ολοκληρωτικά.

Παράδειγμα 1 — n² + n + 41

Παρατηρούμε ότι η έκφραση \(f(n) = n^2 + n + 41\) δίνει πρώτο αριθμό για \(n = 0, 1, 2, \ldots, 39\):

\[ f(0)=41,\; f(1)=43,\; f(2)=47,\; f(3)=53,\;\ldots,\; f(39)=1601. \]

Θα μπορούσαμε να εικάσουμε: «\(f(n)\) είναι πρώτος για κάθε \(n \ge 0\).»

Αντιπαράδειγμα: \(n = 40\):

\[ f(40) = 40^2 + 40 + 41 = 1600 + 40 + 41 = 1681 = 41^2. \]

Σαφώς σύνθετος. Η εικασία καταρρίπτεται.

Παράδειγμα 2 — Ψευδής ισότητα ακέραιων μερών

Υπόθεση: Για κάθε \(n, k \in \mathbb{Z}\) ισχύει \(\left\lfloor \dfrac{n+k}{2} \right\rfloor = \left\lfloor \dfrac{n}{2} \right\rfloor + \left\lfloor \dfrac{k}{2} \right\rfloor\).

Αντιπαράδειγμα: \(n = k = 1\):

\[ \left\lfloor \frac{1+1}{2} \right\rfloor = \lfloor 1 \rfloor = 1, \quad \text{αλλά} \quad \left\lfloor \frac{1}{2} \right\rfloor + \left\lfloor \frac{1}{2} \right\rfloor = 0 + 0 = 0. \]

Η ισότητα αποτυγχάνει πάντα όταν \(n\) και \(k\) είναι και οι δύο περιττοί. Η σωστή ισότητα είναι: \(\lfloor (n+k)/2 \rfloor = \lfloor n/2 \rfloor + \lfloor k/2 \rfloor + \lfloor (n \bmod 2 + k \bmod 2)/2 \rfloor\).

Προσοχή — Ο νόμος των μεγάλων αριθμών δεν βοηθά

Σε ορισμένα προβλήματα, η υπόθεση επαληθεύεται για αδύνατα πολλές περιπτώσεις πριν αποτύχει. Αυτό δεν αποτελεί απόδειξη. Ο συνδυασμός «πολλά παραδείγματα» + «ισχυρή ευρετική επιχειρηματολογία» συνιστά εικασία, όχι θεώρημα.

4. Διάσημες Εικασίες — από Εικασία σε Θεώρημα

Η ιστορία των μαθηματικών είναι γεμάτη προτάσεις που έμειναν ανοιχτά για δεκαετίες ή αιώνες. Μερικές αποδείχθηκαν, άλλες παραμένουν ανοιχτές.

Θεώρημα — Τελευταίο Θεώρημα Fermat (Wiles, 1995)

Εικασία (Fermat, 1637): Δεν υπάρχουν θετικοί ακέραιοι \(x,y,z,n\) με \(n > 2\) τέτοιοι ώστε: \[ x^n + y^n = z^n. \] Ο Fermat έγραψε στο περιθώριο του βιβλίου του: «Έχω μια θαυμαστή απόδειξη, αλλά το περιθώριο είναι πολύ μικρό για να τη χωρέσει.»

Απόδειξη (Andrew Wiles, 1995): 358 χρόνια αργότερα, χρησιμοποιώντας βαθιά αποτελέσματα αλγεβρικής γεωμετρίας (μα\-θη\-μα\-τι\-κά Galois, ελλειπτικές καμπύλες).

Εικασία Goldbach (1742 — ακόμα ανοιχτή)

Κάθε άρτιος ακέραιος \(n > 2\) γράφεται ως άθροισμα δύο πρώτων:

\[ n = p + q, \quad p, q \text{ πρώτοι}. \]

Παράδειγμα: \(4=2+2\), \(6=3+3\), \(8=3+5\), \(100=3+97=11+89\).

Επαληθεύεται υπολογιστικά για όλους τους άρτιους έως \(4 \times 10^{18}\), αλλά δεν έχει αποδειχθεί.

Θεώρημα 4 Χρωμάτων (Appel & Haken, 1976)

Κάθε επίπεδος χάρτης χρωματίζεται με 4 χρώματα ώστε γειτονικές χώρες να έχουν διαφορετικό χρώμα.

Ιστορικά αξιοσημείωτο: πρώτη απόδειξη που χρησιμοποίησε υπολογιστή (ανάλυση ~1.936 περιπτώσεων). Η «αξιοπιστία» ηλεκτρονικής απόδειξης προκάλεσε φιλοσοφική συζήτηση στα μαθηματικά.

Εικασία Collatz / 3x+1 (ακόμα ανοιχτή)

Ορίζουμε:

\[ f(n) = \begin{cases} n/2 & \text{αν } n \text{ άρτιος} \\ 3n+1 & \text{αν } n \text{ περιττός} \end{cases} \]

Εικασία: Εφαρμόζοντας επανειλημμένα \(f\), κάθε θετικός ακέραιος καταλήγει τελικά στο 1.

Παράδειγμα: \(6 \to 3 \to 10 \to 5 \to 16 \to 8 \to 4 \to 2 \to 1\).

Επαληθεύεται για όλα τα \(n \le 2^{68} \approx 2{,}95 \times 10^{20}\). Ο Paul Erdős σχολίασε: «Τα μαθηματικά δεν είναι ακόμα έτοιμα για τέτοια προβλήματα.»

5. P vs NP

Ένα από τα κεντρικά ερωτήματα της Πληροφορικής: είναι εξίσου δύσκολο να λύσουμε ένα πρόβλημα με το να επαληθεύσουμε μια λύση;

Ορισμός — Κλάση P

Η κλάση P (Polynomial time) περιέχει όλα τα γλωσσικά προβλήματα (αποφάσεις «ναι/όχι») που επιλύονται από ντετερμινιστική Μηχανή Turing σε χρόνο \(\mathcal{O}(n^k)\) για κάποιο σταθερό \(k\), όπου \(n = |\text{input}|\).

Παραδείγματα: ταξινόμηση, εύρεση συντομότερης διαδρομής (Dijkstra), πολλαπλασιασμός πινάκων, primality testing (AKS, 2002).

Ορισμός — Κλάση NP

Η κλάση NP (Nondeterministic Polynomial) περιέχει τα προβλήματα για τα οποία κάθε θετική απάντηση («ναι») έχει πολυωνυμικό πιστοποιητικό (certificate) που μπορεί να επαληθευτεί σε \(\mathcal{O}(n^k)\).

Τυπικά: \(L \in \text{NP}\) αν υπάρχει ντετερμινιστική TM \(V\) και πολυώνυμο \(p\) τ.ω.: \[x \in L \iff \exists w,\; |w| \le p(|x|),\; V(x,w) = 1.\] Το \(w\) λέγεται μάρτυρας ή πιστοποιητικό.

Παρανόηση που πρέπει να αποφεύγετε

NP δεν σημαίνει «Non-Polynomial» (μη-πολυωνυμικό). Σημαίνει «Nondeterministic Polynomial». Είναι ανοιχτό αν \(\text{P} = \text{NP}\) ή \(\text{P} \subsetneq \text{NP}\).

5.1 SAT και NP

Ορισμός — SAT (Boolean Satisfiability)

Δίνεται μια τύπος \(\varphi\) Κανονικής Συζευκτικής Μορφής (CNF): \[ \varphi = C_1 \wedge C_2 \wedge \cdots \wedge C_m, \] όπου κάθε clause \(C_i\) είναι διάζευξη literals (\(x_j\) ή \(\neg x_j\)): \[ C_i = \ell_{i,1} \vee \ell_{i,2} \vee \cdots \vee \ell_{i,k_i}. \] Το πρόβλημα SAT ρωτά: υπάρχει ανάθεση τιμών αλήθειας που ικανοποιεί \(\varphi\);

Παράδειγμα: \(\varphi = (x_1 \vee \neg x_2) \wedge (\neg x_1 \vee x_3) \wedge (x_2 \vee \neg x_3)\).

Δοκιμάζουμε \(x_1=T, x_2=T, x_3=T\): \((T \vee F) \wedge (F \vee T) \wedge (T \vee F) = T\). Ικανοποιείται. ✓

Θεώρημα — SAT ∈ NP

Πιστοποιητικό: μια ικανοποιητική ανάθεση τιμών. Επαλήθευση: ελέγχουμε κάθε clause σε χρόνο \(\mathcal{O}(|\varphi|)\).

Άρα SAT \(\in\) NP. (Το αν SAT \(\in\) P είναι ισοδύναμο με P=NP.)

Παράδειγμα — Planar 3-Coloring ∈ NP

Δίνεται επίπεδο γράφημα. Ερώτηση: χρωματίζεται με 3 χρώματα;

Πιστοποιητικό: ο ίδιος ο χρωματισμός. Επαλήθευση: ελέγχουμε κάθε ακμή \((u,v)\) — χρόνος \(\mathcal{O}(|E|)\).

Άρα Planar 3-Coloring \(\in\) NP. (Είναι επίσης NP-πλήρες, αλλά το 4-Coloring επίπεδων γραφημάτων ανήκει στην P λόγω του Θεωρήματος 4 Χρωμάτων!)

5.2 NP-πληρότητα

Ορισμός — NP-πληρότητα

Ένα πρόβλημα \(B\) είναι NP-πλήρες αν:

  1. \(B \in \text{NP}\), και
  2. για κάθε \(A \in \text{NP}\): \(A \le_p B\) (κάθε NP-πρόβλημα ανάγεται πολυωνυμικά στο \(B\)).

Διαισθητικά: τα NP-πλήρη προβλήματα είναι τα «πιο δύσκολα» στην NP.

Θεώρημα Cook-Levin (1971/1973)

Το SAT είναι NP-πλήρες:

  • SAT \(\in\) NP (αποδείχθηκε παραπάνω).
  • Κάθε \(A \in\) NP ανάγεται σε SAT: κατασκευάζουμε από την TM επαλήθευσης ένα κύκλωμα boolean που προσομοιώνει κάθε βήμα της, και μετατρέπουμε σε CNF.

5.3 Αναγωγές (Cook Reductions)

Ορισμός — Πολυωνυμική Αναγωγή

Λέμε ότι το πρόβλημα \(A\) ανάγεται πολυωνυμικά στο \(B\) (γράφουμε \(A \le_p B\)) αν υπάρχει συνάρτηση \(f\) με:

  • \(f\) υπολογίσιμη σε πολυωνυμικό χρόνο,
  • \(x \in A \iff f(x) \in B\).

Σημασία: αν \(B \in \text{P}\), τότε και \(A \in \text{P}\) (αντιθετοαντίστροφη: αν \(A \notin \text{P}\) τότε \(B \notin \text{P}\)).

Παράδειγμα — SAT ≤p Integer Programming

Δίνεται CNF τύπος \(\varphi\) με μεταβλητές \(x_1, \ldots, x_n\). Κατασκευάζουμε ακέραιο πρόγραμμα:

  • Για κάθε μεταβλητή \(x_i\): εισάγουμε \(y_i \in \{0, 1\}\) (\(y_i = 1 \Leftrightarrow x_i = \text{True}\)).
  • Για κάθε clause \(C_j = (\ell_1 \vee \ell_2 \vee \ell_3)\): εισάγουμε ανίσωση: \[ \sum_{\ell_k = x_i} y_i \;+\; \sum_{\ell_k = \neg x_i} (1 - y_i) \;\ge\; 1. \]

Παράδειγμα: για clause \((x_1 \vee \neg x_2 \vee x_3)\):

\[ y_1 + (1 - y_2) + y_3 \ge 1. \]

Η μετατροπή γίνεται σε πολυωνυμικό χρόνο (γραμμικό στο μέγεθος του \(\varphi\)) και \(\varphi\) ικανοποιείται \(\iff\) το ακέραιο πρόγραμμα έχει λύση.

Θεώρημα — Συνέπειες της απάντησης στο P vs NP
Σενάριο Συνέπεια
\(\text{P} = \text{NP}\) Όλα τα NP προβλήματα λύνονται αποδοτικά. Κρυπτογραφία δημόσιου κλειδιού (RSA) καταρρέει. Θεώρηση: «απίθανο».
\(\text{P} \ne \text{NP}\) NP-πλήρη προβλήματα δεν έχουν πολυωνυμικούς αλγόριθμους. Κρυπτογραφία βασισμένη σε NP-δύσκολα προβλήματα παραμένει ασφαλής.
Παράδειγμα — Αλυσίδα NP-πλήρων αναγωγών

Κλασική αλυσίδα (Cook → Karp):

\[ \text{SAT} \le_p \text{3-SAT} \le_p \text{Clique} \le_p \text{Vertex Cover} \le_p \text{Subset Sum} \le_p \cdots \]

Κάθε νέο NP-πλήρες πρόβλημα αποδεικνύεται μέσω αναγωγής από κάποιο ήδη γνωστό.

Συχνές Ερωτήσεις Εξετάσεων

Ερώτηση 1

Τι είναι εικασία και πώς διαφέρει από θεώρημα; Δώστε ένα παράδειγμα.

Προτεινόμενη απάντηση

Μια εικασία είναι πρόταση για την οποία υπάρχουν ενδείξεις αλήθειας αλλά δεν έχει αποδειχθεί ούτε διαψευστεί. Ένα θεώρημα είναι πρόταση που αποδείχθηκε με αυστηρό μαθηματικό επιχείρημα. Παράδειγμα: η Εικασία Goldbach (1742) παραμένει εικασία παρά τις υπολογιστικές επαληθεύσεις, ενώ το Τελευταίο Θεώρημα Fermat έγινε θεώρημα μόλις το 1995.

Ερώτηση 2

Αποδείξτε ότι τα σύνολα \(A = \{\lfloor k\varphi \rfloor : k \ge 1\}\) και \(B = \{\lfloor k\varphi^2 \rfloor : k \ge 1\}\) είναι ξένα (A ∩ B = ∅).

Προτεινόμενη απάντηση

Έστω \(\lfloor p\varphi \rfloor = \lfloor q\varphi^2 \rfloor = m\). Επειδή \(\varphi, \varphi^2\) άρρητοι: \(m < p\varphi < m+1\) και \(m < q\varphi^2 < m+1\). Προσθέτοντας τα κλάσματα: \(m/\varphi + m/\varphi^2 < p+q < (m+1)/\varphi + (m+1)/\varphi^2\), άρα \(m < p+q < m+1\), αδύνατο αφού \(p+q \in \mathbb{Z}\).

Ερώτηση 3

Ορίστε τις κλάσεις P και NP. Πότε ένα πρόβλημα είναι NP-πλήρες;

Προτεινόμενη απάντηση

P: προβλήματα επιλύσιμα σε πολυωνυμικό χρόνο (ντετερμινιστικά). NP: προβλήματα με πολυωνυμικά επαληθεύσιμα πιστοποιητικά. Ένα \(B\) είναι NP-πλήρες αν (i) \(B \in\) NP και (ii) κάθε \(A \in\) NP ανάγεται σε \(B\) πολυωνυμικά (\(A \le_p B\)). Παράδειγμα: SAT (Cook-Levin), 3-SAT, Clique, Vertex Cover.

Ερώτηση 4

Δείξτε ότι SAT ∈ NP. Τι είναι το πιστοποιητικό;

Προτεινόμενη απάντηση

Πιστοποιητικό για στιγμιότυπο με \(n\) μεταβλητές: ανάθεση \(w \in \{0,1\}^n\), μέγεθος \(n = \mathcal{O}(|\varphi|)\). Επαλήθευση: αξιολόγηση κάθε clause — \(\mathcal{O}(|\varphi|)\). Άρα SAT \(\in\) NP.

Ερώτηση 5

Ποιες είναι οι συνέπειες αν αποδειχθεί ότι P = NP;

Προτεινόμενη απάντηση

Αν P = NP: όλα τα NP-πλήρη προβλήματα λύνονται σε πολυωνυμικό χρόνο. Πρακτικές συνέπειες: (α) κρυπτογραφία δημόσιου κλειδιού (RSA, ECC) καταρρέει — η παραγοντοποίηση ακεραίων θα γίνει αποδοτική· (β) βελτιστοποίηση NP-hard προβλημάτων (TSP, scheduling) λύνεται ακριβώς· (γ) θεωρητικά: ολόκληρη η ιεραρχία NP καταρρέει στην P.
