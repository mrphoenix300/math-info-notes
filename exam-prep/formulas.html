✨ Χρυσή Τομή
Ορισμός
\(\varphi = \dfrac{1+\sqrt{5}}{2} \approx 1.618\)
Χαρακτ/κή
\(\varphi^2 = \varphi + 1\)
Συζυγής
\(\hat{\varphi} = \dfrac{1-\sqrt{5}}{2} \approx -0.618\)
Αντίστροφο
\(\dfrac{1}{\varphi} = \varphi - 1\)
Ταυτότητα
\(\dfrac{1}{\varphi} + \dfrac{1}{\varphi^2} = 1\)
🐱 Fibonacci
Αναδρ/κή
\(F_n = F_{n-1} + F_{n-2}\), \(F_1=F_2=1\)
Κλειστός
\(F_n = \dfrac{\varphi^n - \hat\varphi^n}{\sqrt{5}}\) (Binet)
Φράγμα
\(\tfrac{1}{2}\varphi^{n-1} \le F_n \le \varphi^{n-1}\)
Doubling
\(F_{2n} = F_n(2F_{n+1}-F_n)\)
Doubling
\(F_{2n+1} = F_n^2 + F_{n+1}^2\)
Addition
\(F_{n+m} = F_nF_{m+1} + F_{n-1}F_m\)
GCD
\(\gcd(F_m, F_n) = F_{\gcd(m,n)}\)
🔢 Αριθμοθεωρία
Ευκλείδης
\(\gcd(a,b) = \gcd(b,\, a \bmod b)\)
Βάση
\(\gcd(a,0) = a\)
Bézout
\(\gcd(a,b) = \min\{ax+by : ax+by>0,\; x,y\in\Z\}\)
Bézout
\(\exists x,y\in\Z: ax+by = \gcd(a,b)\)
Πολυπλ/τα
Ευκλείδης: \(\le 2\lfloor\log a\rfloor\) βήματα
Fibonacci
Χειρότερη περίπτωση: \(a=F_{n+1}, b=F_n\)
ΘΘΑ
\(a \mid bc \land \gcd(a,b)=1 \Rightarrow a \mid c\)
Πρώτοι
Άπειροι (Ευκλείδης: Υποθέτω πεπερασμένο, πολ/σμός+1)
🎲 Απαρίθμηση & Περιστερεώνας
Pigeonhole
\(n+1\) αντικείμενα σε \(n\) κλάσεις \(\Rightarrow\) κάποια κλάση \(\ge 2\)
Γενικ/μένο
\(m\) αντικ. σε \(n\) κλάσεις \(\Rightarrow\) κλάση \(\ge \lceil m/n \rceil\)
Διωνυμικό
\(\dbinom{n}{k} = \dfrac{n!}{k!(n-k)!}\)
Συμμετρία
\(\dbinom{n}{k} = \dbinom{n}{n-k}\)
Άθροισμα
\(\displaystyle\sum_{k=0}^n\binom{n}{k} = 2^n\)
Dirichlet
\(\exists p/q: \left|\theta - p/q\right| < 1/(mq),\; 1\le q\le m\)
🏭 Γραφήματα
Χειραψία
\(\displaystyle\sum_{v\in V}\deg(v) = 2|E|\)
Δέντρο
\(|E| = |V| - 1\) (συνεκτικός ακυκλικός)
Δάσος
\(k\) συν/νιστώσες: \(|E| = |V| - k\)
Euler κύκλος
Κάθε κόμβος ζυγού βαθμού
Euler μονοπάτι
Ακριβώς δύο κόμβοι περιττού βαθμού
Euler \(n{-}m{+}f\)
\(n - m + f = 2\) (συνεκτικός επίπεδος)
Αραιότητα
\(m \le 3n - 6\) (\(n \ge 3\))
Αραιότητα
\(m \le 2n - 4\) (διμερής, \(n \ge 3\))
Kuratowski
Επίπεδος \(\iff\) χωρίς \(K_5\) ή \(K_{3,3}\) subdivision
Χρωματ.
\(\chi(G) \ge \omega(G)\), διμερής \(\Rightarrow \chi \le 2\)
Επίπεδος
\(\chi(G) \le 6\) (εύκολο), \(\le 4\) (Θ. 4 Χρωμάτων)
🏎 Ramsey
Αναδρ/κή
\(R(k,m) \le R(k{-}1,m) + R(k,m{-}1)\)
Κλειστός
\(R(k,m) \le \dbinom{k+m-2}{k-1}\)
Άνω φράγμα
\(R(k,k) \le \dbinom{2k-2}{k-1} < 4^k\)
Κάτω φράγμα
\(R(t,t) > 2^{t/2}\) (Erdős 1947, πιθανοτική)
Βάσεις
\(R(1,m)=1,\; R(2,m)=m,\; R(3,3)=6\)
Ορισμός
Ελάχιστο \(N\) ώστε κάθε 2-χρωμ. \(K_N\) περιέχει \(K_k\) ή \(K_m\) μονόχρωμο
⚕ Hall, Dilworth, Mirsky, Sperner
Hall
Τέλ. ταίρ. \(\iff \forall S\subseteq L: |N(S)|\ge|S|\)
Defect
\(\max_M |M| = |L| - \delta(G)\), \(\delta=\max_{S}(|S|-|N(S)|)\)
Berge
\(M\) μέγιστο \(\iff\) δεν υπάρχει \(M\)-αυξητικό μονοπάτι
König
\(\nu(G) = \tau(G)\) (διμερής: max matching = min vertex cover)
Dilworth
\(c(P) = \alpha(P)\) (ελ. κάλυψη αλυσίδων = μέγ. αντιαλυσίδα)
Mirsky
Ελ. κάλυψη αντιαλυσίδων = \(\omega(P)\) (μέγ. αλυσίδα)
Sperner
\(|\mathcal{A}| \le \dbinom{n}{\lfloor n/2\rfloor}\) για αντιαλ. \(\mathcal{A}\subseteq 2^{[n]}\)
Intersect.
Τεμνόμενη οικογένεια: \(|\mathcal{F}| \le 2^{n-1}\)
∞ Υπολογισιμότητα & Σύνολα
Cantor
\(\R\) μη-απαριθμήσιμο (διαγωνοποίηση)
Cantor
\(|2^S| > |S|\) για κάθε \(S\)
Απαρ/σιμα
\(\N, \Z, \Q, \Sigma^*\) είναι απαριθμήσιμα
Γλώσσες
\(\mathcal{L} = 2^{\{0,1\}^*}\) μη-απαριθμήσιμο
Halting
Πρόβλημα \(\text{HALT}\) μη-αποφάσιστο
Ιδέα
Πρόγρ. \(D\): «τρέχω εαυτό, αντιστρέφω» → αντίφαση
Rice
Κάθε μη-τετριμμένη σημασιολογική ιδ/τα ΜΤΜ μη-αποφάσιστη
∞ Κλασικά Αθροίσματα & Ανισότητες
\(\sum i\)
\(\displaystyle\sum_{i=1}^n i = \dfrac{n(n+1)}{2}\)
\(\sum i^2\)
\(\displaystyle\sum_{i=1}^n i^2 = \dfrac{n(n+1)(2n+1)}{6}\)
\(\sum i^3\)
\(\displaystyle\sum_{i=1}^n i^3 = \left(\dfrac{n(n+1)}{2}\right)^2\)
\(\sum 2^i\)
\(\displaystyle\sum_{i=0}^n 2^i = 2^{n+1}-1\)
\(H_n\)
\(\displaystyle H_n = \sum_{i=1}^n \frac{1}{i} \ge \frac{\ln(n+1)}{2}\)
AM-GM
\(\dfrac{a+b}{2} \ge \sqrt{ab}\) για \(a,b \ge 0\)
\(n!\) vs \(2^n\)
\(n! > 2^n\) για \(n \ge 4\); \(\; n! > n^{n/2}\) για \(n\ge 3\)
Bernoulli
\((1+x)^n \ge 1+nx\) για \(x \ge -1, n\in\N\)
